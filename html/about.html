HubNet Web is the latest evolution of the HubNet technology for participatory simulations, which typically take the form of something like brief, educational, multiplayer games. HubNet Web re-imagines HubNet as a web application that runs in any modern web browser (including on Chromebooks and smartphones), allows user connections outside of the host's local area network, provides greater flexibility in user roles, and offers a more intuitive authoring experience.

HubNet Web supports all of the simulations from the original HubNet's library of models, and more. These simulations cover topics from social science to physics to biology to mathematics. If none of the activities in the library suit your needs, you may also use the HubNet Web authoring tool to convert a model from NetLogo or NetLogo Web into a HubNet Web activity, or even to build one entirely from scratch.

NetLogo was originally designed by Uri Wilensky and developed at Northwestern University's Center for Connected Learning and Computer-Based Modeling (CCL).
